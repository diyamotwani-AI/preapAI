# PrepAI | Advanced AI-Powered Study Planner

![PrepAI Banner](https://images.unsplash.com/photo-1516321318423-f06f85e504b3?ixlib=rb-1.2.1&auto=format&fit=crop&w=1350&q=80)

**PrepAI** is a premium, student-focused study orchestration platform designed to eliminate planning fatigue. Built with a "Luxury Academic" aesthetic, it uses adaptive logic to generate high-efficiency study roadmaps based on exam dates, difficulty, and user confidence.

## ✨ Key Features

- **🧠 Adaptive AI Insight Engine**: Analyzes study sessions and provides personalized efficiency tips (e.g., "Schedule hard subjects before 12 PM").
- **📊 Real-Time Analytics**: Live tracking of "Days Left," "Total Planned Hours," and "Remaining Work."
- **🔄 Spaced Repetition Logic**: Automatically organizes revision cycles based on the selected difficulty (Beginner, Intermediate, Advanced).
- **💾 Cloud-Free Persistence**: Uses `localStorage` to save user plans directly in the browser—no account required for basic use.
- **🎨 Luxury Design System**: A high-end UI featuring **Deep Almond & Chocolate** color schemes, glassmorphism, and kinetic 3D interactions.
- **📂 Export to PDF**: One-click professional timetable generation for offline use.

## 🛠️ Technology Stack

- **Core**: HTML5 (Semantic), CSS3 (Modern Flex/Grid, Custom Animations)
- **Logic**: Vanilla JavaScript (ES6+)
- **Storage**: Browser localStorage API
- **Design Philosophy**: Minimalist Luxury / Dark Mode Adaptive

## 🚀 How to Use

1. **Enter Details**: Provide your subjects, target exam date, and preferred study intensity.
2. **Generate**: Let the AI orchestrate your daily timeline across study, break, and review slots.
3. **Track**: Use the "Completed today’s study" toggle to watch your progress bar animate in real-time.
4. **Adaptive Response**: Rate your confidence after each session to trigger adaptive learning feedback.
